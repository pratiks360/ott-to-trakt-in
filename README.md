# OTT to Trakt Sync

A modern, dynamic web application designed to safely synchronize OTT viewing histories to your Trakt profile. This powerful utility works with multiple platforms like Netflix, providing fine-grained control and ensuring a persistent sync status.

## Features
- **Multi-Platform Sync**: Easily pull watch history from various OTT platforms.
- **Trakt Integration**: Fast, reliable synchronization to your Trakt account.
- **Safe & Persistent**: Keeps track of what has been synced securely.

## Getting Started

### Prerequisites
- Node.js (v18 or above recommended)
- Trakt API Credentials (Client ID)

### Installation
1. Clone the repository: `git clone https://github.com/pratiks360/ott-to-trakt-in.git`
2. Install the dependencies: `npm install`

### Run Locally
Use Vite's dev server:
```bash
npm run dev
```

### Build
To build for production:
```bash
npm run build
```

## Deployment
Automated deployment is pushed to `gh-pages`.
